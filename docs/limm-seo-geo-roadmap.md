# LIMM SEO and GEO Roadmap

This preview creates the technical and content structure for LIMM Works Pte Ltd. It does not connect the live domain and does not claim migrated Wix content.

## SEO Foundations Included

- Configurable site URL in `src/data/site.ts`
- Page titles and meta descriptions
- Canonical URLs
- Open Graph and Twitter card metadata
- Sitemap via `src/app/sitemap.ts`
- Robots via `src/app/robots.ts`
- JSON-LD for:
  - Organization
  - LocalBusiness
  - WebSite
  - WebPage
  - Service
  - FAQPage
  - BreadcrumbList
  - Article
